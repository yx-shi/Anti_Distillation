# 项目简介
Anti Distillation的目的是找到一种方法，增加大模型蒸馏以得到小模型的难度。核心思路是在保证大模型性能不降的情况下，增大蒸馏的KL距离

# 目前准备做的工作

- 用FSDP将训练加载到多卡进行（已完成）
- 用vLLM跑通inference，包括offline和online两种形式
